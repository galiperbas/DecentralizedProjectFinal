# PoR Blockchain Projesi

Bu projede minimal bir Python blockchain iskeleti üzerine Proof-of-Reputation tabanlı bir konsensüs mekanizması entegre edilmiştir. Mining yerine, doğrulayıcıların reputation skorlarının ortalaması kullanılarak blok kabulü simüle edilmiştir.

## Kurulum ve Çalıştırma

Terminali açın ve şu komutları çalıştırın:

```bash
pip install -r requirements.txt
python -m examples.simple_run
```
